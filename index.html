Methodology & Data Sources

This dashboard combines data from multiple automated sources to track active states of emergency worldwide.

Data Sources:

  • Wikipedia — Active states of emergency scraped from the State of emergency article via the MediaWiki API. This serves as the primary baseline.
  • GDELT DOC 2.0 — Global news monitoring for recent articles mentioning emergency-related terms.
  • ReliefWeb API — Humanitarian reports and active disaster declarations from OCHA/ReliefWeb.

Confidence Score: Each emergency is assigned a confidence score (50–100%) based on the recency and number of confirming sources, agreement between the Wikipedia baseline and live news feeds, and source authoritativeness. Only entries with ≥50% confidence are displayed.

Emergency Types: Entries are automatically classified into five categories — Disaster, Public Health, Conflict/Security, Migration, and Governance — using keyword matching against titles and descriptions.

Update Frequency: The data pipeline runs automatically every 6 hours via GitHub Actions.

Limitations: This dashboard relies on automated data collection and may miss emergencies not covered by Wikipedia or English-language news sources. Classification is keyword-based and may occasionally miscategorize entries. Emergency start dates may be approximate.
